    - Tự động lưu các thư viện liên quan **(tất cả) (fix: Chỉ lưu các các thư viện phụ thuộc trực tiếp thoi)**
    
  - Homepage:
    - Hiển thị những version chỉnh sửa gần đây 
    - Tìm kiếm project
